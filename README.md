# Flutter_Simple_Frontend
A usual Assignment for college

------------------------------------------------------------
HOW TO INSTALL AND RUN
------------------------------------------------------------

STEP 1: Create the Flutter Project
Open your terminal and run:
> flutter create storage_lab_app

STEP 2: Replace the Main Code
1. Navigate to the 'lib' directory.
2. Open 'main.dart'.
3. Delete all existing code.
4. Paste the provided StorageLabApp code into this file.

STEP 3: Fix the Widget Test (Optional but Recommended)
To ensure your project passes tests without errors:
1. Navigate to the 'test' directory.
2. Open 'widget_test.dart'.
3. Find the line: 
   await tester.pumpWidget(const ExampleApp());
4. Replace "ExampleApp" with "StorageLabApp" to match the main class name.

STEP 4: Run the App
Connect your device/emulator and run:
> flutter runp"
